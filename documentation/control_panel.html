Super Search contains some tools and controls in the ExpressionEngine control panel:

  • Homepage
  • Search Log
  • Fields
  • Utilities
  • Preferences
  • Demo Templates
  • Documentation

Homepage

This tab displays a list of the Top Search Terms and Recent Searches. Clicking on a search term will then give you further information about that search term.

Search Log

This tab displays a list of all recent searches and details about them. Clicking on a search term will then give you further information about that search term. The Log Site Searches? preference in the Super Search control panel must be set to Yes in order for this page to show results.

Fields

Super Search includes a utility that lets admins alter their custom channel fields at the MySQL level. This is important since you can improve site performance by changing field types to use only the space you need. Example, if you have a field that always contain either y or n, change it to a char field and save tons of space. The most important part of the Field Manager is that you can change some fields to numeric types. For example, people often have a price field. They should change that EE custom field to a decimal field with precision of 2 so that searches on that field will order and sort properly.

Utilities

This tab contains a utility called Build Search Lexicon, which is used to build a combined dataset of all unique terms across your sites, which enables fuzzy searches, search corrections and better search term handling. The first run may take some time, but only needs to run once.

Preferences

This tab contains a variety of settings you can adjust to enable/disable search behaviours and features, including cache preferences.

Demo Templates

If you wish to load the demo templates for Super Search, go to this tab. Then just create a prefix name for the sample data such as test_ or super_search_, and click the Install Demo Templates button.

Documentation

All documentation is available on the Solspace site. This tab will take you directly to the Solspace site documentation area.
