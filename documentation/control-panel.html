Control Panel

Super Search contains some tools and controls in the ExpressionEngine control panel:

  • Top Search Keywords
  • Search Log
  • Search Curation
  • Build Search Lexicon
  • Preferences
  • Demo Templates

Top Search Keywords

This page displays a list of the Top Search Keywords as well as some information about them.

Search Log

This page displays a list of all recent searches and details about them. The Log Site Searches? preference in the Super Search control panel must be set to Yes in order for this page to show results.

Search Curation

Super Search includes a Search Curation feature, which is designed to allow manual selection of entry results for keyword searches. This would typically be used as supplemental search results to normal search results, similar to how Google displays sponsored results at the top or side of search results pages. Follow the interface steps to use this feature.

Build Search Lexicon

Super Search contains a utility called Build Search Lexicon, which is used to build a combined dataset of all unique terms across your sites, which enables fuzzy searches, search corrections and better search term handling. The first run may take some time, but only needs to run once.

Preferences

This page contains a variety of settings you can adjust to enable/disable search behaviors and features, including cache preferences.

Demo Templates

If you wish to load the demo templates for Super Search, go to this tab. Then just create a prefix name for the sample data such as test_ or super_search_, and click the Install Demo Templates button.
